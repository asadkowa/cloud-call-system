🔧 SIP Configuration

🌐 Server Configuration

Server/Host: 192.168.1.38
Port: 5060
Transport: UDP
Domain: 192.168.1.38.pbx.local

🔐 Account Setup

To get your extension credentials:

  1. Open web browser: http://192.168.1.38:3004
  2. Login to your account
  3. Go to Extensions page
  4. Create or select a "user" extension
  5. View extension details for SIP credentials

🔧 App Configuration Steps

📱 Linphone Setup

  1. Open Linphone app
  2. Go to Settings → Accounts → Add Account
  3. Select SIP Account
  4. Enter your credentials:
    Username: [Your Extension Number]
    Password: [SIP Password from web interface]
    Domain: 192.168.1.38.pbx.local
    Server: 192.168.1.38:5060
    Transport: UDP
  5. Save and register

📱 Zoiper Setup

  1. Open Zoiper app
  2. Add Account → SIP
  3. Enter configuration:
    Account Name: Extension [Number]
    Host: 192.168.1.38
    Username: [Your Extension Number]
    Password: [SIP Password from web interface]
  4. Save configuration

🧪 Testing

✅ Test Extension Calling

  1. Create 2 extensions (e.g., 1001, 1002) via web interface
  2. Configure 2 mobile devices with different extensions
  3. Check registration status in the web interface
  4. Make a call from one mobile app to the other extension number
  5. Monitor call logs in the web dashboard

🔍 Troubleshooting

Common Issues

  • Registration Failed: Check network connection and credentials
  • Can't reach server: Ensure phone and computer are on same WiFi
  • No audio: Check firewall settings and app permissions
  • Calls don't connect: Verify both extensions are registered

📞 Quick Test

Once configured, dial the other extension number directly in your SIP app to test calling between extensions!
